# 应用设置

## 常规设置

| 设置项 | 说明 |
|--------|------|
| 语言 | 中文 / English |
| 主题 | 跟随系统 / 浅色 / 暗色 |
| 启动显示欢迎引导 | 首次启动显示引导教程 |

## 图像设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 默认格式 | 选择默认的代码生成格式 | AutoJs找色 |
| 放大镜刷新间隔 | 50-1000ms | 200ms |

## 快捷键设置

点击「快捷键」查看所有可配置的快捷键。

## 导入/导出设置

- **导出**：将当前设置导出为 JSON 文件
- **导入**：从 JSON 文件导入设置
- **恢复默认**：重置所有设置为默认值

## 日志

点击「打开日志文件夹」查看应用日志，便于排查问题。

## 设置文件位置

配置文件存储在用户目录下：
- **Windows**：`%USERPROFILE%\.qi-qi-multi-point-color-finder\settings.yaml`
- **macOS**：`~/.qi-qi-multi-point-color-finder/settings.yaml`
