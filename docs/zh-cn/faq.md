# 常见问题

## Q: 提示「未找到 adb」

**A**: 请确保已安装 ADB 并配置环境变量。参考本文档「安装与环境配置」章节。

```cmd
# 验证 ADB 是否安装
adb version
```

## Q: 连接失败

**A**: 检查以下事项：
1. 手机和电脑在同一网络
2. 手机已开启 USB 调试
3. 端口未被占用（默认 5555）
4. 防火墙未阻止连接

**排查步骤**：
```cmd
# 查看设备列表
adb devices

# 尝试连接
adb connect 192.168.1.100:5555
```

## Q: 截图失败

**A**:
1. 确认设备已连接
2. 尝试重新连接设备
3. 检查手机屏幕是否可正常显示
4. 查看日志文件排查具体原因

## Q: 颜色匹配不准

**A**:
1. 降低相似度设置（如从 100% 降到 95%）
2. 确保截图清晰
3. 尝试使用锚点模式提高稳定性
4. 检查颜色点标记位置是否准确

## Q: 代码复制失败

**A**:
1. 检查剪贴板权限
2. 尝试重新生成代码后复制
3. 手动选中代码文本复制

## Q: 应用无法启动

**A**:
1. 检查是否已注册许可证
2. 查看日志文件：`~/.qi-qi-multi-point-color-finder/logs/`
3. 尝试恢复默认设置

## Q: 快捷键冲突

**A**:
在设置中更改快捷键为不冲突的按键组合。
