# 界面介绍

## 顶部栏

| 区域 | 说明 |
|------|------|
| 设备选择 | 下拉框选择已连接/历史设备 |
| 连接按钮 | 连接/断开设备 |
| 截图按钮 | 获取设备屏幕截图 |

## 左侧面板

| 区域 | 说明 |
|------|------|
| 格式选择 | 选择代码生成格式 |
| 相似度 | 设置颜色匹配相似度 |
| 缩放控制 | 调整图像缩放比例 |
| 操作按钮 | 导入、截图、旋转、清空 |

## 中间图像区域

| 区域 | 说明 |
|------|------|
| 主画布 | 显示当前截图/导入图片 |
| 颜色点 | 已标记的颜色点显示 |
| 框选区域 | 拖拽选中的目标区域 |

## 右侧面板

| 区域 | 说明 |
|------|------|
| 颜色列表 | 所有已添加的颜色点 |
| 放大镜 | 鼠标悬停位置的高清放大视图 |
| 状态栏 | 测试结果、坐标信息 |
