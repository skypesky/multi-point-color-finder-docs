# 获取屏幕截图

## ADB 截图

点击「截图」按钮，通过 ADB 获取当前设备屏幕。如果目标设备不在连接列表中，会自动尝试连接。

**快捷键**：`S`

## 图片导入

支持多种方式导入本地图片：

| 方式 | 操作 |
|------|------|
| 文件导入 | 点击「导入」按钮选择图片 |
| 拖放 | 直接拖拽图片文件到应用窗口 |

**支持格式**：PNG、JPG、BMP

## 多 Tab 管理

- 每次截图/导入自动创建新 Tab（无数量限制）
- 点击 Tab 切换，长按关闭

## 图像旋转

点击「旋转」按钮将图像顺时针旋转 90°。

**快捷键**：`R`

## 保存图片

点击保存按钮可将当前图像保存为 PNG 文件。
