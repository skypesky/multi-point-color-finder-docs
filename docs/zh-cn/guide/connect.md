# 连接设备

## 添加新设备

1. 点击设备下拉框底部的「添加设备」
2. 输入设备 IP 地址和端口号
3. 点击「连接」

**常见端口**：`5555`（ADB WiFi 默认端口）

## 连接状态说明

| 状态图标 | 含义 |
|---------|------|
| 🟢 绿点 | 设备已连接，可执行截图 |
| ⚪ 灰点 | 设备离线，未连接 |
| 🔴 红点 | 连接失败 |

## 批量连接

启动应用时，会自动探测最近连接过的设备的在线状态（无数量限制）。

## 断开设备

点击已连接设备旁边的「×」按钮可断开连接并从列表移除。

## 自动重连

当设备离线后重新上线，点击截图时会自动尝试连接，无需手动重新添加。

## 截图时自动连接

当点击截图按钮时，如果目标设备不在连接列表中，会自动尝试连接该设备。无需手动添加设备即可直接截图。
