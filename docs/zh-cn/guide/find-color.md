# 测试与找色

## 测试流程

1. 确保已添加至少一个颜色点
2. （可选）设置框选区域缩小搜索范围
3. 点击「测试」按钮
4. 查看结果：
   - **找到**：`找到: (x, y)` 并在图像上高亮显示位置
   - **未找到**：`未找到`

**快捷键**：`T`

## 相似度设置

可设置颜色匹配相似度：

| 相似度 | 说明 |
|--------|------|
| 100% | 完全匹配 |
| 95% | 允许轻微偏差 |
| 90% | 中等偏差容忍 |
| 85% | 较高偏差容忍 |
| 80% | 高偏差容忍 |

## 测试结果说明

### 找到

当颜色匹配成功时，显示：
- 找到的坐标位置 `(x, y)`
- 图像上高亮显示位置（持续约 3 秒）

### 未找到

当所有相似度下都无法匹配时，显示「未找到」。

**建议**：降低相似度设置或确保截图清晰。
