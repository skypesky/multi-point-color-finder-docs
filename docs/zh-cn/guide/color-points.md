# 标记颜色点

## 添加颜色点

### 方式一：点击图像

- 点击图像上的任意位置
- 自动获取该像素的坐标和颜色值

### 方式二：手动输入

- 在输入框中直接输入 X、Y 坐标
- 点击「添加」按钮

## 锚点类型

每个颜色点可设置锚点类型：

| 类型 | 说明 |
|------|------|
| None | 普通颜色点 |
| L | 左锚点 |
| C | 中心锚点 |
| R | 右锚点 |

## 颜色列表操作

| 操作 | 说明 |
|------|------|
| 点击序号 | 切换该点是否参与生成代码 |
| 点击坐标 | 复制 `x,y` 到剪贴板 |
| 点击颜色值 | 复制十六进制颜色如 `#RRGGBB` |
| 删除按钮 | 删除该颜色点 |

## 清空列表

点击「清空」按钮移除所有已标记的颜色点。

**快捷键**：`F1`
