# 快速指引

## 前提条件

开始之前，请确保已完成以下准备：

1. **安装应用** — 按照 [Windows 安装指南](install/windows) 或 [macOS 安装指南](install/macos) 安装本应用
2. **安装 ADB** — Android Debug Bridge 是连接 Android 设备的必要工具，参见安装指南中的 ADB 配置
3. **连接设备** — 确保 Android 设备已开启 USB 调试并通过 ADB 连接到电脑

## 5 步完成找色脚本

### 步骤 1：连接设备

1. 在顶部设备下拉框中选择设备，或点击「添加设备」输入 `IP地址:端口号`（如 `192.168.1.100:5555`）
2. 点击「连接」按钮
3. 状态栏显示「已连接」表示连接成功

![连接设备](/images/device-connect.png)

### 步骤 2：截图

1. 确保设备已连接
2. 点击「截图」按钮获取设备当前屏幕
3. 截图自动显示在中间图像区域

### 步骤 3：标记颜色点

1. 点击图像上的目标位置，自动添加颜色点
2. 在右侧颜色列表中设置锚点类型（None/L/C/R）
3. 可点击颜色值复制十六进制颜色码

![标记颜色点](/images/color-points-marked.png)

### 步骤 4：测试

1. 设置完成后点击「测试」按钮
2. 查看返回结果：「找到: (x, y)」或「未找到」

![测试结果](/images/test-result.png)

### 步骤 5：生成代码

1. 选择代码格式（AutoJs找色/AutoJs比色/大漠找色/锚点找色/锚点比色）
2. 点击「生成」按钮
3. 点击「复制」将代码复制到剪贴板

![生成代码](/images/code-gen-result.png)
