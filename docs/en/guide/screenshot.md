# Screenshot

## Screenshot Methods

### ADB Screenshot (Default)

1. Ensure device is connected
2. Click "Screenshot" button to capture current screen
3. Screenshot appears in the center image area

![Screenshot Button](../../images/guide-screenshot-button.png)

### Local Import

1. Click "Import" or drag image file directly
2. Supports PNG, JPG, BMP formats

### Clipboard Paste

1. Copy image to clipboard
2. Press `Ctrl+V` / `Cmd+V` to paste

### Drag & Drop

1. Drag image file onto the application window
2. Image will be loaded automatically

## Notes

- Screenshot will be saved in tabs, you can switch between multiple tabs
- ADB screenshot requires device connection
- For local import/clipboard/drag & drop, no device connection needed
