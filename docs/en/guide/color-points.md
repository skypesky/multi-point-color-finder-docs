# Color Points

## Add Color Points

Click anywhere on the screenshot to add a color point. The point will appear as a numbered marker on the image.

![Add Color Point](/multi-point-color-finder-docs/images/guide-color-add-point.png)

## Color Point Properties

Each color point has:

- **Position**: X, Y coordinates on the image
- **Color**: Hex color value (e.g., `#FF5733`)
- **Anchor Type**: None, L (Left), C (Center), R (Right)

## Anchor Types

| Type | Description |
|------|-------------|
| None | Regular reference point |
| L | Left anchor - used as primary reference |
| C | Center anchor - used for position calculation |
| R | Right anchor - secondary reference |

## Edit Color Points

- **Move**: Drag the color point to a new position
- **Delete**: Right-click → Delete or select and press Delete key
- **Copy Color**: Click the color value to copy hex code

## Color List

Right panel shows all color points with:
- Point number
- X, Y coordinates
- Hex color value
- Anchor type

You can reorder points and batch set anchor types.
